# Changelog

## [0.1.0] - 2026-02-28

### Added
- Full syntax highlighting for E++ (.epp) files
- Hover tooltips for built-in keywords (say, let, set, define, repeat, while, etc.)
- Auto-completion snippets for common E++ constructs
- One-click Run button to execute E++ programs directly from the editor
- Support for pseudo code syntax (DECLARE, PRINT, FUNCTION, IF/THEN, FOR/TO, WHILE)
- Basic block-balancing diagnostics
